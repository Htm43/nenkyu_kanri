
# 年休管理アプリ　設計書

## 全体構成

```
アプリ本体     → GitHub Pages（静的ホスティング）
データ(JSON)  → 個人のOneDriveフォルダ
               形式は今のJSONのまま
```

---

## PCブラウザ版

**現状のまま、変更なし**

```
GitHub PagesのURLをブラウザで開く
↓
「ファイルを開く」でOneDrive内のJSONを選ぶ
↓
読み書き・自動保存（今と同じ）
↓
OneDriveが勝手にクラウド同期
```

---

## モバイルアプリ版（Android / iOS）

**新規開発**

```
GitHubのリリースページにAPKを置く（Android）
↓
直インストール
↓
OneDrive内の同じJSONを読み書き
↓
PCで編集したデータをスマホで確認・編集
```

### 技術構成

```
Flutterで書く（共通）
├── アプリ本体のUI
├── JSON読み書き
├── OneDrive連携
└── ビジネスロジック全部

ネイティブで書く（追加するだけ）
├── Androidウィジェット（Kotlin）
└── iOSウィジェット（Swift）
```

### 実装する機能

| 機能 | 優先度 |
|------|--------|
| OneDriveのJSONを開く | 必須 |
| サマリー表示（残り時間・日数・期限） | 必須 |
| 記録の追加・編集・削除 | 必須 |
| 前回ファイルの自動復元 | 必須 |
| ホーム画面ウィジェット | 後回し |
