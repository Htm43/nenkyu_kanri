# 年次有給休暇 取得管理

ブラウザで使う、年休取得の記録ツールです。

- 初回: JSONファイルを新規作成して保存先を決める
- 2回目以降: 前回ファイルを自動復元（権限が残っている場合）
- 入力後は自動保存（手動保存ボタン不要）
- 設定は「付与総日数」で入力し、内部で「1日あたり時間」から時間換算

## ファイル構成

- Nenkyu.html: 画面の構造
- Nenkyu.css: スタイル
- Nenkyu.js: アプリロジック（表示、計算、保存、復元）
- nenkyuu.json: データファイル例

## 使い方

1. Nenkyu.html を Edge/Chrome で開く
2. 初回は「新規作成」を押して JSON の保存先を選ぶ
3. 取得データを入力する
4. 変更のたびに JSON へ自動保存される

## 保存仕様

- File System Access API を使用
- 前回のファイルハンドルを IndexedDB に保存
- 起動時に前回ハンドルの権限が有効なら自動で読み込み
- 権限が失効している場合は「前回ファイル復元」または「開く」から再選択

## 注意

- このツールは File System Access API を使うため、file:// 直開きの環境では挙動が制限される場合があります。
- 動作が不安定な場合は、ローカルサーバー経由（例: VS Code Live Server）で開いてください。
